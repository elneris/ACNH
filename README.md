# ACNH : Trade Navets

Outils pour faciliter les échanges entre joueurs lors du cours du navets

Développer par Elnéris

- Php 7.3 / Symfony 4.4

## Init

- Cloner le projet
- docker-compose up -d
- composer install
- php bin/console server:run

## Architecture

- Env de prod : https://ac-market.fr/
- Env de dev : https://dev.ac-market.fr/
